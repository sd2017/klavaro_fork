Documentation

Le manuel pour la version récente (1.0.8) n'est pas encore traduit en français. Pour l'instant, nous disposons uniquement des versions en portugais et en esperanto.

Si vous voulez aider le project Klavaro en traduisant la documentation en français, consultez ce document source qui utilise LaTeX: klavaro-dok-1.0.8.tar.gz. Vous devez comprendre soit le portugais soit l'esperanto pour traduire à partir d'une de ces deux langues.

Forums / Discussions

Si vous voulez nous écrire, si vous avez une suggestion ou besoin d'aide, veuillez visiter les forums sur SourceForge. Vous y trouverez des espaces dédiés en plusieurs langues : anglais, esperanto, portugais, etc.

Pour poster sur le forum, vous devrez disposer d'un compte sur SourceForge. Cette condition est requise pour éviter le spam anonyme. Si vous n'avez pas encore de compte sur SourceForge, vous pouvez en créer un ici.
