Documentation

The last version of the manual (1.0.8) is not translated to English. For now, there are only the Portuguese and Esperanto versions.

If you want to help this project on translating its documentation to English, see this source file, it uses LaTeX: klavaro-dok-1.0.8.tar.gz. Then, you are supposed to translate from Portuguese or from Esperanto, of course.

Forums / Discussions

If you want to give us some feedback, sugestion, ask for help, support or any thing else, please visit the forums at SourceForge. There you will find rooms for English, Esperanto, Portuguese, etc.

Be aware that in order to post there, you must have an account at SourceForge. That is done to avoid spamming from anonymous people. If you still are not registered, please spend some minutes first doing it: create an account at SourceForge.
